# Momus
Momus is a cross-platfrom python tool that designed for anti-censorship image processing.

Library:
```
PySimpleGUI
numpy
cv2
matplotlib
pillow
```

Install:
```bash
pip install PySimpleGUI numpy opencv-python  matplotlib pillow
```
run:
```bash
python3 Momus.py
```
Or use release version
