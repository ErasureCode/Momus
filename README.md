# Momus
Momus is a cross-platfrom python tool that designed for anti-censorship image processing.

Library:
```
PySimpleGUI
numpy
cv2
time
image_processing
matplotlib
PIL
```

Install:
```bash
pip install PySimpleGUI numpy cv2 time image_processing matplotlib PIL
```
run:
```bash
python3 Momus.py
```
Or use release version
